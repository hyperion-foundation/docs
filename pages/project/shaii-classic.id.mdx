## Informasi Project
Shaii Classic merupakan sebuah Discord bot musik dan dapat digunakan secara gratis.

### Versi Project/Bot
`v1.0.0`

## Tujuan Project
Shaii Classic tidak pernah dibuat untuk menggantikan Discord bot musik lainnya. Kami membuat semacam project ini karena kami ingin membuat sesuatu yang dapat digunakan semua orang dan bebas untuk digunakan. Kami tidak pernah memaksa para pengguna/anggota kami untuk membayar kami untuk menggunakan layanan kami, tetapi Anda dipersilakan untuk mendukung layanan kami dan menjaga layanan kami tetap hidup. Jika anda berkenan untuk meng-support project/layanan kami, silahkan baca informasinya lebih lanjut di [halaman ini](/id/support).

## Cara mengundang Shaii Classic
Klik link berikut ini untuk mengundang Shaii Classic ke server anda : [Undang Shaii Classic](https://bit.ly/shaii-classic)

## Cara menggunakan Shaii Classic
Ketik `sc!help` atau `sc!commands` untuk melihat daftar perintah Shaii Classic. Kami menaruh banyak informasi pada setiap perintah, jadi anda dapat memahami fungsi-fungsi perintah tersebut.

## Bot Permission
Shaii Classic tidak memerlukan permission yang bersifat **administratif**. Berikut di bawah ini adalah daftar permission yang dibutuhkan Shaii Classic dan aktifkan permission ini dan Shaii Classic akan berfungsi dengan baik : 
1. General Server Permissions
  - View Channel
2. Voice Channel Permissions
  - Connect
  - Speak
  - Use Voice Activity
3. Text Channel Permissions
  - Send Messages and Create Posts
  - Embed Links
  - Read Message History
