## Project Information
Pandora is a server security Discord bot. The bot is not available for public.

#### Project/Bot Version
`v0.2.0`

## Project Goal
We made Pandora for securing our Discord servers from bad people.

## How to Invite Shaii Classic
Currently, there is no such way to invite Pandora to your server. We'll try to consider to make another version of Pandora if Pandora is a successful project.

## How to Use Pandora
Pandora commands is only available for Hyperion Foundation staff. All Hyperion Foundation members cannot use Pandora commands, because all Pandora commands are for administrative functionality.
