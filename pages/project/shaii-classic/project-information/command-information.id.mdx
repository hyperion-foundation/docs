# Informasi Perintah Shaii Classic
Setiap perintah Shaii Classic memiliki beberapa informasi. Informasi ini seharusnya cukup bagi Anda untuk mempelajari perintah tersebut. Pada bagian ini, kami mencantumkan informasi perintah yang terperinci dan paling sederhana.

- **Command Description (en-US)**
  - Informasi ini menampilkan deskripsi perintah yang terperinci dan paling sederhana dalam **Bahasa Inggris**.
- **Command Description (id)**
  - Informasi ini menampilkan deskripsi perintah yang terperinci dan paling sederhana dalam **Bahasa Indonesia**.
- **Command Cooldown**
  - Informasi ini menampilkan berapa lama setiap perintah dapat digunakan kembali setelah dijalankan.
- **Command Availability**
  - Informasi ini menampilkan ketersediaan perintah.
    - Jika ketersediaannya **Ya**, yang berarti perintah tersebut tersedia untuk digunakan.
    - Jika ketersediaannya **Tidak**, yang berarti perintah tersebut tidak tersedia atau tidak dapat digunakan. Mungkin sedang dalam pengembangan, bug, dll.
- **Voice Channel Requirement**
  - Informasi ini ditampilkan jika perintah ini memerlukan koneksi saluran suara pengguna.
    - Jika persyaratannya **Diperlukan**, pengguna harus berada di saluran suara terlebih dahulu sebelum menjalankan perintah.
    - Jika persyaratannya **Tidak Diperlukan**, pengguna tidak perlu masuk ke saluran suara apa pun untuk menjalankan perintah.
- **Localization**
  - Informasi ini menampilkan perintah dalam bahasa lain berdasarkan bahasa yang Anda atur di aplikasi Discord Anda. Saat ini, kami hanya mendukung Bahasa Inggris (`en-US`) & Bahasa Indonesia (`id`).
- **Example Usage**
  - Informasi ini menampilkan cara menggunakan perintah.
