## Project Information
Shaii Classic is a public general-purpose Discord music bot and it's free to use.

## Project/Bot Version
`v2.0.0-beta.1`

## Project Goal
Shaii Classic never made a goal to replaced other Discord music bots. We made kind of this project because we want to create something that can be used for all people and free to use. We never force our users/members to pay us to use our services, but you are so welcome to support us and keeping our services alive.

## How to Invite Shaii Classic
There are 2 ways to invite Shaii Classic to your server:
1. Invite [Shaii Classic](https://discord.com/oauth2/authorize?client_id=703520871838515230&permissions=36777024&integration_type=0&scope=bot) **with** pre-assigned client permissions. \*
2. Invite [Shaii Classic](https://discord.com/oauth2/authorize?client_id=703520871838515230&permissions=0&integration_type=0&scope=bot) **without** pre-assigned client permissions.

\* *We recommend you to use this method, because you don't need to manually assign any permissions, but after you authorize it, the Shaii Classic role cannot be removed.*

## How to Use Shaii Classic
Type `/help` to see the list of Shaii Classic commands.

## Bot/Client Permissions
If you choose the second method, you can assign these permissions to a role or directly for Shaii Classic. Shaii Classic doesn't need an administrative permissions. This is the list of permissions that Shaii Classic needs and enable these permissions and the bot should be functioning properly : 
1. General Server Permissions
  - View Channel
2. Voice Channel Permissions
  - Connect
  - Speak
  - Use Voice Activity
3. Text Channel Permissions
  - Send Messages
  - Add Reactions
  - Read Message History
