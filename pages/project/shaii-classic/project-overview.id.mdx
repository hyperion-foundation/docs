## Informasi Project
Shaii Classic merupakan sebuah bot musik Discord dan dapat digunakan secara gratis.

## Versi Project/Bot
`v2.0.0-beta.1`

## Tujuan Project
Shaii Classic tidak pernah dibuat untuk menggantikan bot musik Discord lainnya. Kami membuat semacam project ini karena kami ingin membuat sesuatu yang dapat digunakan semua orang dan bebas untuk digunakan. Kami tidak pernah memaksa para pengguna/anggota kami untuk membayar kami untuk menggunakan layanan kami, tetapi Anda dipersilakan untuk mendukung layanan kami dan menjaga layanan kami tetap hidup. Jika anda berkenan untuk meng-support project/layanan kami, silahkan baca informasinya lebih lanjut di [halaman ini](/id/support).

## Cara mengundang Shaii Classic
Ada 2 cara untuk mengundang Shaii Classic ke server anda:
1. Undang [Shaii Classic](https://discord.com/oauth2/authorize?client_id=703520871838515230&permissions=36777024&integration_type=0&scope=bot) dengan izin/permission yang telah ditetapkan sebelumnya. \*
2. Undang [Shaii Classic](https://discord.com/oauth2/authorize?client_id=703520871838515230&permissions=0&integration_type=0&scope=bot) tanpa izin/permission yang telah ditetapkan sebelumnya.

> \* *Kami menyarankan anda untuk pilih metode ini, karena anda tidak perlu untuk menetapkan izin/permission apa pun secara manual, tetapi setelah anda mengotorisasinya, peran/role Shaii Classic tidak dapat dihapus.*

## Cara menggunakan Shaii Classic
Ketik `/help` untuk melihat daftar perintah Shaii Classic.

## Izin Klien/Bot
Shaii Classic tidak memerlukan permission yang bersifat administratif. Berikut di bawah ini adalah daftar permission yang dibutuhkan Shaii Classic dan aktifkan permission ini dan Shaii Classic akan berfungsi dengan baik : 
1. General Server Permissions
  - View Channel
2. Voice Channel Permissions
  - Connect
  - Speak
  - Use Voice Activity
3. Text Channel Permissions
  - Send Messages
  - Add Reactions
  - Read Message History
