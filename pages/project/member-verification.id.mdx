## Informasi Project
Member Verification merupakan sebuah web project untuk memverifikasi akun para member/anggota Hyperion di server Discord Hyperion. Project ini masih dalam tahap pengerjaan.

## Tujuan Project
Project ini kami buat hanya untuk memverifikasikan member/anggota Hyperion apabila akun Discord tersebut merupakan akun yang sah dan bukan akun self-bot.

## Cara verifikasi akun
Project ini belum ter-realisasikan dan belum ada tanggal pasti kami akan merilis project ini.
