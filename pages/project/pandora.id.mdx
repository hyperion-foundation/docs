## Informasi Project
Pandora adalah bot Discord pribadi untuk mengamankan server Discord kami dari aktivitas berbahaya.

#### Versi Project/Bot
`v0.2.0`

## Tujuan Project
Pandora sengaja dibuat untuk mengamankan server Discord kami. Pandora dapat memindai dan menghapus kata-kata buruk, dll.

## Cara mengundang Pandora
Pandora bukan bot Discord publik seperti [Shaii Classic](/project/shaii-classic). Anda tidak dapat mengundang Pandora ke server Anda.
Kami akan mencoba mempertimbangkan untuk membuat versi lain dari Pandora yang dapat digunakan untuk umum.

## Self host
Jika Anda tertarik, Anda dapat menghosting Pandora secara mandiri. Kunjungi [repositori Pandora](https://github.com/hyperion-foundation/pandora) untuk meng-clone (mengunduh) source codenya.

Namun perlu diketahui, source code Pandora saat ini tidak terlalu stabil (dev), itu karena kami masih mengembangkannya (fitur minimal). Jika Anda masih bersikeras untuk menghostingnya secara mandiri, harap gunakan source codenya dengan menanggung resiko secara mandiri.

Kami TIDAK akan memberikan dukungan apa pun jika Anda memiliki masalah di pihak Anda.

## Cara menggunakan Pandora
Semua anggota Hyperion Foundation hanya dapat menggunakan perintah dasar Pandora seperti melihat versi project/bot, dll.
Perintah administratif (misalnya ban member, kick member, dll) tidak dapat digunakan jika anggota tidak memiliki peran (role) dan izin (permission) yang diperlukan.
