## Informasi Project
Pandora merupakan sebuah bot Discord untuk mengamankan server. Bot ini tidak tersedia untuk publik.

#### Versi Project/Bot
`v0.2.0`

## Tujuan Project
Kami membuat Pandora untuk menjaga server-server Discord kami dari orang-orang jahat.

## Cara mengundang Pandora
Untuk saat ini, tidak ada cara untuk mengundang Pandora ke server anda. Kami akan coba mempertimbangkan untuk membuat versi lain dari Pandora jika Pandora merupakan sebuah project yang sukses dan stabil.

## Cara menggunakan Pandora
Perintah bot Pandora hanya tersedia untuk staf Hyperion Foundation. Semua member Hyperion Foundation tidak bisa menggunakan perintah Pandora, karena semua perintah Pandora untuk fungsi administratif.
