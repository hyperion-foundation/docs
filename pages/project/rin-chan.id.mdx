## Informasi Project
Rin-chan adalah bot musik Discord pribadi. Rin-chan dibuat dan dihosting secara eksklusif untuk server **Hyperion Foundation** Discord.
Playlist musik Rin-chan disediakan oleh [Lofi Girl](https://www.youtube.com/@LofiGirl).

### Versi Project/Bot
Proyek jenis ini tidak memiliki versi apa pun, karena tidak akan ada perubahan pada proyek.

## Cara Mengundang Rin-chan
Sayangnya, Rin-chan bukanlah bot Discord publik seperti [Shaii Classic](/project/shaii-classic), jadi Anda tidak dapat mengundang Rin-chan ke server Anda.

## Cara menggunakan Rin-chan
Rin-chan tidak memiliki perintah publik seperti yang dimiliki oleh [Shaii](/project/shaii) atau [Shaii Classic](/project/shaii-classic), jadi Anda tidak dapat berinteraksi dengan Rin-chan.
