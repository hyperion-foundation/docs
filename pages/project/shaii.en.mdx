## Project Information
Shaii is a different edition of Shaii Classic. Shaii is made for Hyperion Discord server only (non-public Discord bot).

## Project Goal
Shaii purposely made for Hyperion Discord server only, that's it. Shaii is no different with Shaii Classic, it's just a different project name.

## How to Invite Shaii
Shaii is not a public Discord bot, so you can't have Shaii on your server.

## How to Use Shaii
Type `s!help` or `s!commands` in **#bot-commands** (Hyperion Discord server) to see the list of Shaii commands.
