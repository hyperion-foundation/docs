## Project Information
Shaii Classic is a public general-purpose Discord music bot and it's free to use.

#### Project/Bot Version
`v1.0.0`

## Project Goal
Shaii Classic never made a goal to replaced other Discord music bots. We made kind of this project because we want to create something that can be used for all people and free to use. We never force our users/members to pay us to use our services, but you are so welcome to support us and keeping our services alive.

## How to Invite Shaii Classic
Click this link to invite Shaii Classic to your server : [Invite Shaii Classic](https://bit.ly/shaii-classic)

## How to Use Shaii Classic
Type `sc!help` or `sc!commands` to see the list of Shaii Classic commands. We put a lot of information on each command, so you can understand what each command is used for.

## Bot Permissions
Shaii Classic doesn't need an administrative permissions. This is the list of permissions that Shaii Classic needs and enable these permissions and the bot should be functioning properly : 
1. General Server Permissions
  - View Channel
2. Voice Channel Permissions
  - Connect
  - Speak
  - Use Voice Activity
3. Text Channel Permissions
  - Send Messages and Create Posts
  - Embed Links
  - Read Message History
