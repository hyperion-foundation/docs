## Informasi Project
Shaii merupakan edisi yang berbeda dari [Shaii Classic](/id/project/shaii-classic). Shaii dibuat hanya untuk server Discord Hyperion saja (non-public Discord bot).

#### Versi Project/Bot
`v2.1.2`

## Tujuan Project
Shaii dibuat dengan tujuan hanya untuk server Discord Hyperion saja. Shaii tidak berbeda dengan [Shaii Classic](/id/project/shaii-classic), hanya penamaan projectnya saja yang berbeda.

## Cara mengundang Shaii
Shaii bukanlah sebuah Discord bot publik seperti [Shaii Classic](/id/project/shaii-classic), jadi anda tidak bisa mengundang Shaii ke server anda.

## Cara menggunakan Shaii
Ketik perintah `s!help` atau `s!commands` di channel **#bot-commands** (Hyperion Discord server) untuk melihat daftar perintah Shaii.

Fitur **Slash Command** di Shaii sudah tersedia walaupun masih ada beberapa perintah yang belum sempurna untuk digunakan. Ketik `/help` di channel **#bot-commands** (server Discord Hyperion Foundation) untuk melihat daftar perintah Shaii.
