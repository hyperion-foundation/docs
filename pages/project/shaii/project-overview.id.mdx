## Deskripsi Proyek
Shaii merupakan edisi yang berbeda dari Shaii Classic. Shaii dibuat hanya untuk server Discord **Hyperion Foundation** saja (non-public Discord bot).

#### Versi Proyek/Bot
`v2.0.1`

## Tujuan Proyek
Shaii dibuat dengan tujuan hanya untuk server Discord **Hyperion Foundation** saja. Shaii tidak berbeda dengan Shaii Classic, hanya penamaan proyeknya saja yang berbeda.

## Cara mengundang Shaii
Shaii bukanlah sebuah Discord bot publik seperti **Shaii Classic**, jadi anda tidak bisa mengundang Shaii ke server anda.

## Cara menggunakan Shaii
Ketik perintah `/help` di channel **#bot-commands** (**Hyperion Foundation** Discord server) untuk melihat daftar perintah Shaii.

Fitur **Perintah Slash** kini menjadi sistem interaksi default. Kami telah menonaktifkan fitur **legacy command** secara permanen. (Fitur ini dinonaktifkan pada **10 Desember 2023**)
