# Log Perubahan Shaii
Ini adalah log perubahan publik untuk **Shaii**. Semua perubahan Shaii publik akan dicantumkan di halaman ini.
> Jika perubahannya terlalu kecil, kami tidak akan menulis perubahan apa pun di halaman ini.

Catatan: Ada beberapa perubahan lama yang tidak dapat dicantumkan di halaman ini.

### Versi `1.0.0` - 20 Januari 2023
- Rilis publik awal.

### Versi `1.3.5` - 31 Agustus 2023
- [Changed]
   - Menambahkan pernyataan kondisi ketika pengguna tidak dapat memasukkan angka kurang dari 1 (volume rendah).
   - Menambahkan pernyataan kondisi ketika pengguna tidak dapat memasukkan angka lebih dari 100 (earrape).

### Versi `1.4.0` - 15 Oktober 2023
- [Added]
   - Menambahkan suatu event **addList** dan menonaktifkan Legacy Command untuk sementara.
   - Menambahkan **jump** dan **lirik** ke perintah **help**.
   - Menambahkan fungsi pemilihan untuk mencegah pengguna memutuskan sambungan bot jika bot tidak berada di saluran suara pada perintah **keluar**. (*Fungsi ini masih dalam pengembangan*)
   - Menambahkan respons pesan jika pengguna hanya meminta lagu/musik pada perintah **play**.
   - Menambahkan alias perintah pada perintah **queue**.

- [Changed]
   - Pembaruan konten pesan kecil pada perintah **volume**.
   - Perintah **play**, **repeat** & **volume** untuk Slash Command tersedia untuk digunakan.
   - Mengubah konten respons pesan pada perintah **queue**.

- [Fixed]
   - Memperbaiki **kesalahan permintaan lagu (daftar putar)**.

### Versi `1.5.0` - 23 Oktober 2023
