Discord Member Verification

All information about verification system in Hyperion Discord server.

Why I should get verified?

We want to ensure if you are a real human, not a bot. There are some hacked Discord accounts and those accounts has been used for phising, etc. And we don't want any bot/hacked accounts exist in our Discord server.

Discord OAuth2

We're using Discord.js library for verification system. We never store any sensitive information of your Discord account in our server such as passwords, etc. Learn more about our Privacy Policy.
